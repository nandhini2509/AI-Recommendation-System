# AI Recommendation System

## Project Description

This project is a simple AI Recommendation System developed using Python. It recommends items based on the user's selected interest category.

## Features

- Interest-based recommendations
- Multiple recommendation categories
- Easy-to-use menu interface
- Continuous interaction until user exits

## Categories Available

- Technology
- Sports
- Music
- Movies
- Books

## Technologies Used

- Python 3

## Files Included

- recommendation_system.py
- code_screenshot.png
- output_screenshot.png
- README.md

## How to Run

1. Install Python.
2. Save the program as recommendation_system.py.
3. Open a terminal in the project folder.
4. Run:

   python recommendation_system.py

5. Enter an interest category to receive recommendations.
